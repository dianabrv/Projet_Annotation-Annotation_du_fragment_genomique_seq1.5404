# Projet_Annotation-Annotation_du_fragment_genomique_seq1.5404
Ce projet vise à annoter un fragment génomique d'origine inconnue, supposé provenir de la famille des Bacillota. Le dépôt regroupe les fichiers de sortie des logiciels d'analyse, les résultats détaillés ainsi que deux programmes de parsing développés spécifiquement pour extraire et traiter les données issues de ce flux de travail.
